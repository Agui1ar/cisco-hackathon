hello solution makers

Welcome to Yes-k, our solution for a real-time personnel management system capable of using a real-time database, cloud storage, communication with the work team using cisco spark and thanks to meraki it is possible to locate the workers within the internal network of the company.

To solve the challenge we leverage the google firebase platform which, in addition to being highly scalable, offers storage, real-time database, among other products that work together using a web console that allows developers total control over the system in general.
